VinAnalytics Public API

Escalabilidad comercial por ecosistemas (OE2). Marketplaces, distribuidores e integradores consumen la inteligencia vitivinícola directamente desde sus plataformas. Diseñada bajo Specification-Driven Development — contrato en /api/v1/openapi.json.

🔑 Autenticación

Todas las solicitudes (excepto /health) requieren una API key en el header X-API-Key (o el parámetro ?api_key=). Límite de tasa: {{ rate_limit }} solicitudes/minuto por key.

curl -H "X-API-Key: {{ demo_key }}" \
     "{{ request.host_url }}api/v1/mercados"

🧭 Endpoints

GET/api/v1/healthpúblico

Estado del servicio y nº total de reseñas disponibles.

GET/api/v1/vinos🔒 API key

Catálogo de vinos paginado con filtros.

ParámetroTipoDescripción
pageintPágina (default 1)
per_pageintTamaño (máx 100)
paisstringFiltrar por país
variedadstringFiltrar por variedad
puntos_minintPuntuación mínima
GET/api/v1/mercados🔒 API key

Indicadores agregados por mercado (reseñas, puntuación y precio promedio).

GET/api/v1/precios🔒 API key

Recomendaciones de precio dinámico por variedad (modelo de IA).

GET/api/v1/scorecard🔒 API key

Resumen ejecutivo del Balanced Scorecard (MRR, churn, uptime, clientes activos).
